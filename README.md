![](commad.png)
> A modern version of `cd` from now on.


## Recomended
```sh
: cd
alias ,='commad'

: prevd
alias ,,=', -1'
alias ,,,=', -2'
alias ,,,,=', -3'
alias ,,,,,=', -4'
alias ,,,,,,=', -5'

: nextd
alias ,,=', -1'
alias ,.=', +1'
alias ,..=', +2'
alias ,...=', +3'
alias ,....=', +4'
alias ,.....=', +5'

: cd parents
alias ..=', ..'
alias ...=', ../..'
alias ....=', ../../..'
alias .....=', ../../../..'
alias ......=', ../../../../..'
alias ~=', ~'
```
